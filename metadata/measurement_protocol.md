# Measurement protocol summary

The source files describe multimodal personal monitoring sessions collected in July 2021. Each session may include cuff-based blood pressure, pulse oximetry, ECG PDF report, phonocardiographic WAV recording, environmental measurements and contextual activity notes. Sleep and step-count variables are daily-level variables attached to session-level observations.

Device models, calibration status and exact measurement posture were not specified in the source files and should be documented separately if available.
